# Skill Name
Phân tích Quản lý Tài sản

## Level
Level 2 - Business Skill

## Purpose
Phân tích vòng đời của tài sản doanh nghiệp (Enterprise Asset Management).

## Inputs
- Danh mục tài sản
- Chính sách khấu hao

## Process
1. Mua sắm tài sản
2. Ghi tăng tài sản & Cấp phát
3. Bảo trì bảo dưỡng
4. Khấu hao
5. Thanh lý (Ghi giảm)

## Outputs
- Luồng Quản lý tài sản
- Cấu trúc dữ liệu Asset

## Checklist
- [ ] Có tạo mã vạch/QR code cho mỗi tài sản không?
- [ ] Có theo dõi lịch sử luân chuyển tài sản giữa các phòng ban không?
