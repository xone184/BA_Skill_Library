# Skill Name
Tạo Template Meeting Minutes

## Level
Level 1 - Basic Skill

## Purpose
Sinh ra biểu mẫu Biên bản họp (MOM) chuyên nghiệp.

## Inputs
- Tên cuộc họp
- Thành phần tham dự

## Process
1. Header cuộc họp
2. Mục tiêu (Agenda)
3. Nội dung trao đổi
4. Action Items (Người phụ trách, Deadline)

## Outputs
- Biểu mẫu MoM

## Checklist
- [ ] Mọi Action Item đều phải có người chịu trách nhiệm rõ ràng (Assignee) và Deadline.
- [ ] Có phần kết luận chốt vấn đề chưa?
