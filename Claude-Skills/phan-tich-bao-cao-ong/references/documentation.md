# Skill Name
Phân tích Báo cáo động

## Level
Level 3 - Architecture Skill

## Purpose
Thiết kế hệ thống báo cáo cho phép người dùng tự kéo thả (Dynamic Reporting).

## Inputs
- Nhu cầu phân tích dữ liệu đa chiều

## Process
1. Xác định Dimension (Thuộc tính phân nhóm: Thời gian, Khu vực)
2. Xác định Measure (Chỉ số đo lường: Doanh thu, Số lượng)
3. Thiết kế giao diện Pivot/Kéo thả

## Outputs
- Cấu trúc khối dữ liệu (Data Cube)
- Danh sách Dimension và Measure

## Checklist
- [ ] Có hỗ trợ Export ra Excel/PDF giữ nguyên định dạng không?
- [ ] Có phân quyền dữ liệu theo cấp độ (Row-level security) không?
