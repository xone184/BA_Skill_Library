# Skill Name
Thiết kế Form

## Level
Level 2 - Business Skill

## Purpose
Phân tích và tối ưu hóa các Form nhập liệu.

## Inputs
- Danh sách các trường cần nhập

## Process
1. Phân nhóm trường dữ liệu (Grouping)
2. Sắp xếp theo thứ tự ưu tiên
3. Định nghĩa Validation rules (Required, Format, Min/Max)
4. Thiết kế thông báo lỗi

## Outputs
- Bố cục Form (Wireframe)
- Luật Validation
