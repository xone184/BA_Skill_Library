# Skill Name
Class Diagram

## Level
Level 3 - Architecture Skill

## Purpose
Vẽ biểu đồ lớp để thiết kế hướng đối tượng.

## Inputs
- Danh từ trong tài liệu Yêu cầu
- Quy tắc nghiệp vụ

## Process
1. Xác định Classes
2. Định nghĩa Attributes (Thuộc tính)
3. Định nghĩa Methods (Hành động)
4. Vẽ Relationships (Association, Inheritance, Aggregation)

## Outputs
- Class Diagram (Mermaid)

## Checklist
- [ ] Có lạm dụng Inheritance (Kế thừa) thay vì Composition không?
- [ ] Visibility (Public/Private) đã rõ ràng chưa?
