# Skill Name
Xây dựng kịch bản UAT

## Level
Level 2 - Business Skill

## Purpose
Chuẩn bị kịch bản User Acceptance Testing cho người dùng cuối.

## Inputs
- Quy trình nghiệp vụ thực tế
- BRD

## Process
1. Lựa chọn các quy trình cốt lõi
2. Chuẩn bị dữ liệu mẫu (Test Data)
3. Viết các bước thao tác trên góc độ User

## Outputs
- UAT Script theo luồng (End-to-End)

## Checklist
- [ ] Ngôn ngữ đã dễ hiểu với End-user chưa?
- [ ] Dữ liệu test có giống thực tế không?
