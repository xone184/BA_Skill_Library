# Skill Name
Tạo Template RTM

## Level
Level 1 - Basic Skill

## Purpose
Tạo ma trận truy xuất yêu cầu (Requirement Traceability Matrix).

## Inputs
- Danh sách BRD
- Danh sách Test Case

## Process
1. Tạo cột Requirement ID
2. Tạo cột Use Case ID
3. Tạo cột Design ID
4. Tạo cột Test Case ID
5. Tạo cột Status

## Outputs
- Bảng RTM chuẩn

## Checklist
- [ ] Có Requirement nào bị 'mồ côi' (Không có Test Case) không?
- [ ] Tất cả ID đã được map 1-1 hoặc 1-N chưa?
