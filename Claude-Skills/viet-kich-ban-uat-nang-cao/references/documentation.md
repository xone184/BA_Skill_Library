# Skill Name
Viết kịch bản UAT nâng cao

## Level
Level 2 - Business Skill

## Purpose
Xây dựng tài liệu hướng dẫn khách hàng ký nghiệm thu (Sign-off).

## Inputs
- UAT Script đã review
- Tiêu chí nghiệm thu

## Process
1. Cung cấp tài khoản test
2. Hướng dẫn khách hàng thao tác
3. Ghi nhận phản hồi
4. Phân loại Bug (Lỗi hệ thống) vs CR (Thay đổi yêu cầu)

## Outputs
- Tài liệu UAT Sign-off form
- Hướng dẫn ghi nhận lỗi (Bug tracking)

## Checklist
- [ ] Đã thống nhất định nghĩa BUG vs CR trước khi khách hàng test chưa?
- [ ] Có giới hạn số vòng UAT không?
