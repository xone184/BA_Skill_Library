# Skill Name
Sequence Diagram

## Level
Level 3 - Architecture Skill

## Purpose
Vẽ biểu đồ tuần tự để thiết kế logic tương tác giữa các hệ thống.

## Inputs
- Use Case
- Danh sách Actor và System

## Process
1. Định nghĩa Lifelines
2. Vẽ thông điệp gọi hàm (Call Message)
3. Vẽ thông điệp trả về (Return Message)
4. Định nghĩa các khối điều kiện (Alt/Opt/Loop)

## Outputs
- Sequence Diagram (Mermaid)

## Checklist
- [ ] Có xử lý ngoại lệ (Exception/Error) trên diagram không?
- [ ] Có Timeout/Retry logic không?
