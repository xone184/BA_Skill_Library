# Skill Name
Phân tích Business Rules

## Level
Level 3 - Architecture Skill

## Purpose
Trích xuất và chuẩn hóa các quy tắc nghiệp vụ rải rác.

## Inputs
- Quy trình nghiệp vụ
- Chính sách công ty

## Process
1. Đọc tài liệu nghiệp vụ
2. Trích xuất các câu điều kiện (If-Then)
3. Tổ chức thành Decision Table hoặc Decision Tree
4. Gán ID cho từng rule để trace

## Outputs
- Bảng Business Rules Catalog
- Decision Table (Bảng quyết định)

## Checklist
- [ ] Có Rule nào mâu thuẫn với Rule khác không?
- [ ] Có Rule nào chưa có cách xử lý (Missing Else) không?
