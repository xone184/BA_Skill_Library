# Skill Name
Data Flow Analysis

## Level
Level 3 - Architecture Skill

## Purpose
Phân tích luồng di chuyển của dữ liệu qua các hệ thống.

## Inputs
- Sơ đồ kiến trúc hiện tại
- Quy trình nghiệp vụ

## Process
1. Xác định External Entities
2. Vẽ luồng DFD Level 0 (Context)
3. Vẽ luồng DFD Level 1 (Chi tiết)
4. Xác định nơi lưu trữ dữ liệu

## Outputs
- Data Flow Diagram (DFD)
- Danh sách Data Store

## Checklist
- [ ] Có dữ liệu nào đi vào mà không đi ra không (Blackhole)?
- [ ] Có dữ liệu nào tự sinh ra không có nguồn gốc không (Miracle)?
