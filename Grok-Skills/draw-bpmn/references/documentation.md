# Skill Name
Draw BPMN

## Level
Level 2 - Business Skill

## Purpose
Vẽ luồng quy trình nghiệp vụ bằng chuẩn BPMN.

## Inputs
- Mô tả quy trình bằng lời

## Process
1. Xác định các Pool/Lane (Actors)
2. Xác định điểm bắt đầu (Start Event)
3. Liệt kê các Task
4. Xác định các điểm rẽ nhánh (Gateway)
5. Kết thúc (End Event)

## Outputs
- Sơ đồ BPMN (định dạng PlantUML/Mermaid)

## Checklist
- [ ] Có thiếu Gateway (Exclusive/Parallel) không?
- [ ] Luồng có đi đến End Event không?
