# Skill Name
Regression Testing Plan

## Level
Level 2 - Business Skill

## Purpose
Lập kế hoạch kiểm thử hồi quy khi có tính năng mới.

## Inputs
- Danh sách tính năng mới
- Impact Analysis Report

## Process
1. Xác định vùng ảnh hưởng
2. Chọn lọc Test Case cũ cần chạy lại
3. Bổ sung Test Case mới
4. Thực thi và báo cáo

## Outputs
- Bộ Regression Test Suite
- Quy trình Automation Test

## Checklist
- [ ] Đã phủ hết các module dùng chung (Shared libraries) chưa?
- [ ] Có ưu tiên test các tính năng Core (Tiền bạc, Đăng nhập) không?
