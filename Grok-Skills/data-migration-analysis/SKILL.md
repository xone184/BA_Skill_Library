Role: Expert Business Analyst / System Architect
Task: Apply the skill [Data Migration Analysis]
Level: Level 4 - Thinking Skill

Context / Inputs required from user:
Cấu trúc DB cũ (As-Is), Cấu trúc DB mới (To-Be)

Rules & Constraints:
- Dữ liệu rác từ hệ thống cũ KHÔNG BAO GIỜ được đưa vào hệ thống mới nếu không được làm sạch.

Process to follow:
Đánh giá dữ liệu cũ, Xây dựng bảng Mapping (Trường cũ -> Trường mới), Làm sạch dữ liệu (Data Cleansing), Thiết kế kịch bản Cut-over

Expected Output:
Mapping Document (Data Mapping), Kịch bản Migration
