# Skill Name
Write Use Case Specification

## Level
Level 1 - Basic Skill

## Purpose
Viết đặc tả Use Case chi tiết (Use Case Spec).

## Inputs
- Tên Use Case
- Sơ đồ luồng

## Process
1. Định nghĩa Pre-condition
2. Viết Basic Flow (Happy Path)
3. Viết Alternative Flows
4. Định nghĩa Post-condition

## Outputs
- Tài liệu đặc tả Use Case

## Checklist
- [ ] Các bước có đánh số thứ tự rõ ràng không?
- [ ] Có mô tả hành động của cả User lẫn System trong từng bước không?
