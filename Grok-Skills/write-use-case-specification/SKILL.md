Role: Expert Business Analyst / System Architect
Task: Apply the skill [Write Use Case Specification]
Level: Level 1 - Basic Skill

Context / Inputs required from user:
Tên Use Case, Sơ đồ luồng

Rules & Constraints:
- Cú pháp mỗi bước luôn là: '1. Hệ thống hiển thị...', '2. User nhập...'.

Process to follow:
Định nghĩa Pre-condition, Viết Basic Flow (Happy Path), Viết Alternative Flows, Định nghĩa Post-condition

Expected Output:
Tài liệu đặc tả Use Case
