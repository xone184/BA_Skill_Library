# Skill Name
Risk Analysis

## Level
Level 4 - Thinking Skill

## Purpose
Phân tích và quản lý rủi ro dự án / hệ thống.

## Inputs
- Kiến trúc hệ thống
- Kế hoạch triển khai

## Process
1. Nhận diện rủi ro
2. Đánh giá Xác suất (Likelihood) & Mức độ ảnh hưởng (Impact)
3. Lập kế hoạch giảm thiểu
4. Kế hoạch dự phòng (Contingency)

## Outputs
- Risk Register (Ma trận rủi ro)
- Kế hoạch ứng phó (Mitigation Plan)

## Checklist
- [ ] Có phương án Rollback nếu deploy thất bại không?
- [ ] Có đánh giá rủi ro pháp lý/bảo mật không?
