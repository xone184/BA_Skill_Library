# Skill Name
Tạo Template Use Case Spec

## Level
Level 1 - Basic Skill

## Purpose
Tự động sinh ra biểu mẫu chuẩn cho Use Case Specification.

## Inputs
- Tên Use Case
- Tác nhân

## Process
1. Tạo phần Header (Name, Actor, Goal)
2. Tạo bảng luồng chính (Main Flow)
3. Tạo bảng luồng phụ (Alt Flow)
4. Phần điều kiện trước/sau

## Outputs
- Cấu trúc file Markdown chuẩn

## Checklist
- [ ] Có chừa sẵn không gian cho Business Rules không?
- [ ] Có phần lịch sử sửa đổi không?
