# Skill Name
Database Naming Convention

## Level
Level 3 - Architecture Skill

## Purpose
Định nghĩa quy chuẩn đặt tên cho Database.

## Inputs
- Các thực thể nghiệp vụ

## Process
1. Chuẩn hóa tên Bảng (số ít/số nhiều)
2. Quy định tiền tố/hậu tố
3. Quy định tên Khóa chính/Khóa ngoại
4. Quy định Index naming

## Outputs
- Quy ước đặt tên Bảng, Cột, Khóa

## Checklist
- [ ] Đã cấm sử dụng Reserved Words chưa?
- [ ] Có quy định độ dài tối đa cho tên cột không?
