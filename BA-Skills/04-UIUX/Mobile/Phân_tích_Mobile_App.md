# Skill Name
Phân tích Mobile App

## Level
Level 3 - Architecture Skill

## Purpose
Phân tích nghiệp vụ và UI/UX đặc thù cho nền tảng Mobile.

## Inputs
- Nghiệp vụ cần mang lên Mobile

## Process
1. Xác định chức năng cốt lõi cho Mobile
2. Thiết kế luồng thao tác 1 tay
3. Xử lý trạng thái Offline/Mất mạng
4. Quản lý Push Notification

## Outputs
- Sơ đồ luồng màn hình Mobile
- Thiết kế Bottom Navigation

## Checklist
- [ ] Có tính năng quét QR/Barcode không?
- [ ] Có lưu Cache offline không?
