# Skill Name
Phân tích Security Testing

## Level
Level 3 - Architecture Skill

## Purpose
Xác định các lỗ hổng bảo mật và viết kịch bản test an toàn thông tin.

## Inputs
- Kiến trúc hệ thống
- Tài liệu API

## Process
1. Phân tích Authorization/Authentication
2. Phân tích rủi ro Injection (SQL, XSS)
3. Viết kịch bản kiểm thử bảo mật
4. Lên kế hoạch Pentest

## Outputs
- Security Test Cases (OWASP Top 10)

## Checklist
- [ ] API có Rate Limiting không?
- [ ] Mật khẩu có được băm (Hashed) trước khi lưu không?
