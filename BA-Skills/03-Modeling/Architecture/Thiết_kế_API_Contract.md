# Skill Name
Thiết kế API Contract

## Level
Level 3 - Architecture Skill

## Purpose
Thiết kế đặc tả API cơ bản giữa FE và BE hoặc với bên thứ 3.

## Inputs
- Yêu cầu tích hợp
- Dữ liệu cần truyền

## Process
1. Xác định Endpoint & Method
2. Định nghĩa Request Header/Body
3. Định nghĩa Response (Success/Error)
4. Xác định mã lỗi HTTP

## Outputs
- Tài liệu API Contract (RESTful)

## Checklist
- [ ] Có Token Authentication không?
- [ ] Có phân trang (Pagination) đối với list không?
